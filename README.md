# 📊 Data Professional Survey Dashboard

## 📌 Project Overview

This project is an interactive **Power BI dashboard** built to analyze survey responses from data professionals around the world. The dashboard transforms raw survey data into meaningful insights about salaries, programming languages, job roles, and job satisfaction in the data industry.

The goal of this project is to demonstrate **data analytics, visualization, and KPI development skills using Power BI**.

---

## 🛠 Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Data Cleaning & Transformation** – Preparing survey data for analysis
* **KPI Analysis** – Identifying key metrics from survey responses

---

## 📊 Key KPIs

The dashboard highlights several important metrics from the survey:

* **Total Number of Survey Respondents**
* **Average Age of Respondents**
* **Average Salary by Job Role**
* **Favorite Programming Languages**
* **Job Satisfaction Levels**
* **Work-Life Balance Ratings**

---

## 📈 Dashboard Features

* Interactive visualizations
* Salary comparison across different data roles
* Programming language popularity analysis
* Job satisfaction and work-life balance insights
* Demographic overview of survey participants

## 🔍 Key Insights

Some insights observed from the dataset include:

* Certain data roles report significantly higher average salaries.
* **Python** appears as one of the most popular programming languages among respondents.
* Work-life balance satisfaction varies across different roles in the data industry.
* Survey respondents come from a diverse range of age groups and professional backgrounds.
